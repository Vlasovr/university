Инструкция по использованию приложения:

1. Компиляция:
   - Для работы приложения с использованием готовых тестов, необходимо скомпилировать его в конфигурации x64.
   - Каталог tests и файл stat.dat должны находится в корне проекта (либо на одном уровне с .exe)

2. Администраторский доступ:
   - Для доступа к меню администратора требуется файл kts_key.dat. 
   - Файл kts_key.dat должен быть размещен на съемном носителе.

3. Альтернативный способ:
   - Если вы не хотите использовать kts_key.dat, откройте файл Main.cpp.
   - На строке 47 замените KeyVerifier::verify() на true.
